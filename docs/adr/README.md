# Registros de Decisiones de Arquitectura (ADR)

Este directorio contiene las Decisiones de Arquitectura para Cortex Knowledge Assistant.

## ¿Qué es un ADR?

Un ADR (Architecture Decision Record) es un documento corto que captura una decisión de arquitectura importante junto con su contexto y consecuencias.

## Formato

Cada ADR sigue la estructura:

1. **Contexto**: La situación que requiere una decisión
2. **Decisión**: La elección que se tomó
3. **Alternativas**: Opciones consideradas
4. **Consecuencias**: Impacto de la decisión
